# BookLore

BookLore is an immersive digital bookshelf built with Next.js, React, and TypeScript.

Books are displayed on a virtual shelf where hovering over a spine reveals subtle depth and motion. Selecting a book opens a dedicated page containing summaries, author information, reviews, and purchase links. As the user scrolls, a realistic 3D book animation rotates and gradually opens to reveal its pages, creating an experience inspired by physical books.

## Live Demo

[View the project](https://anshuraikwar.github.io/BookLore/)

## Features

- Interactive bookshelf interface
- Hover-driven 3D book spine animations
- Dedicated pages for each book
- Scroll-synchronized book opening animation
- Responsive layout
- Rich typography and dark theme
- Built with Next.js, React, TypeScript and CSS 3D transforms

## How to run

1. Install dependencies: `npm install`
2. Start the development server: `npm run dev`
3. Open [http://localhost:3000](http://localhost:3000) in your browser
